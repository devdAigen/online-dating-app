# Online Dating App ❤️

An Android-based online dating application that helps users discover and connect with people based on shared interests. Users can register, create and manage their profiles, receive personalized profile recommendations, chat anonymously, and form matches through mutual interest.

## Features

- 👤 **User Registration**
  - Register using basic personal information.
  - Secure user account creation and authentication.

- 📝 **Profile Management**
  - Edit profile information.
  - Add personal details, interests, and preferences.
  - Keep profile information up to date.

- 🎯 **Personalized Recommendations**
  - View profiles recommended based on shared interests.
  - Browse potential matches.

- ❤️ **Mutual Match System**
  - Like or express interest in other users.
  - A match is created automatically when two users like each other.

- 💬 **Anonymous Chat**
  - Users can communicate anonymously before revealing their identities.
  - Encourages safe and comfortable conversations.
  - Identity remains hidden until users choose to disclose it.

- 🤝 **Match List**
  - View all mutual matches.
  - Access matched users from a dedicated match list.

## Project Workflow

1. User registers and creates an account.
2. User completes their profile and selects interests.
3. The application recommends profiles with similar interests.
4. Users express interest in other profiles.
5. When two users mutually express interest, they become a match.
6. Matched users can start an anonymous conversation.
7. Users can update their profiles and manage their matches at any time.
